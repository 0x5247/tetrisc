registers that are used as global variables:
s0 = row position
s1 = column position
s2 = orientation
s3 = block type
s4 = time step for block dropping
s5 = time step
s6 = lines completed
s7 = score
s8 = aaab bb00 0000 0000 0000 0000 000c defg
	- a = current level
	- b = level ups through line completion
	- c = text showing the level needs update
	- d = text showing the lines needs update
	- e = score for hiding next block
	- f = instruction is shown
	- g = next block is hidden
s9 = grid state
s10= blocks grid offsets
s11= block string

t0 = grid string
t1 = hex digits addy
t2 = fixed blocks string
t3 = random numbers addy
t4 = random numbers idx
t5 = time delay addy
t6 = next block
